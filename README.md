# Decidim::Metabase

Display your Metabase dashboards directly in Decidim's administration area.

## Usage

Metabase will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-metabase"
```

And then execute:

```bash
bundle
```
